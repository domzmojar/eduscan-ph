Setup (one-time):

  1. Open Google Sheets and create a new spreadsheet.
  2. Click Extensions → Apps Script.
  3. Delete any existing code and paste the script below.
  4. Click Deploy → New deployment → Web app. Set "Who has access" to Anyone. Copy the Web App URL.
  5. Paste that URL in the field below and tap Save URL.
