📋 Google Sheets Setup Guide

Follow these steps once per class:

1
Create a Google Sheet — Name it after the class (e.g. "Grade 7 Sampaguita Attendance")
2
Open Extensions → Apps Script in your Google Sheet
3
Paste this code and click Save:
function doGet(e) {
  var sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();
  var p = e.parameter;
  sheet.appendRow([
    new Date(), p.student,
    p.studentId, p.mode,
    p.className
  ]);
  return ContentService
    .createTextOutput("OK");
}
4
Click Deploy → New Deployment
5
Set Execute as: Me, access: Anyone
6
Copy the Web App URL and paste it into EduScan PH
